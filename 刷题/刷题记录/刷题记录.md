## 左右指针

### 三数之和

+ **题目**

  > 给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，`使得 a + b + c = 0`？请你找出所有和为 0 `且不重复`的三元组。
  >
  > > ```
  > > 输入：nums = [-1,0,1,2,-1,-4]
  > > 输出：[[-1,-1,2],[-1,0,1]]
  > > ```

+ **解法**

  > 1. 数组排序( 从小到大 )
  >
  > 2. 指定 `nums[i]`, 此时 **L 指针**为 `nums[i+1]`, **R 指针**为 `nums[nums.length-1]`
  >
  >    > <img src="%E5%88%B7%E9%A2%98%E8%AE%B0%E5%BD%95.assets/Screenshot%20from%202021-04-22%2021-49-49.png" alt="Screenshot from 2021-04-22 21-49-49" style="zoom:150%;" />
  >
  > 3. sum = nums[i] + nums[L] + nums[R]
  >
  >    > 1. 如果 **sum<0,** 则 L 指针前进一位, `因为 L 前进才可能使 sum 增大`
  >    > 2. 否则 R 后退一位, `因为 R 后退才可能使 sum 减小`
  >    > 3. 如果 **L 和 R 相遇, 或者 L > R**, 则 i 指针向前一位, 并且重新赋值 L 和 R 指针. `因为 L >= R, 表示 num[i] 的情况无解`
  >
  >    > 1. 如果 **sum == 0**, 则 L 指针前进一位, R 指针后退一位, `在更小的范围寻找解`
  >    >    1. 如果此时 nums[L] == nums[L+1], 应该 L++. `因为三元组中, 确定两个数则确定唯一的解`
  >    >    2. 如果此时 nums[R] == nums[R-1], 应该 R--. `理由同上`
  >    > 2. L 和 R 的其他限制条件与上同
  >
  >    > 1. 如果 **nums[i]==nums[i-1]**, 说明`结果会重复`, 此时应该 i++, 并且重新赋值 L 和 R 指针. `因为每次指定 nums[i] 都会寻找全部可能解`
  >    > 2. 如果 **nums[i] > 0**, 应该直接退出. `因为数组是排序的, 这种情况代表无解.`
